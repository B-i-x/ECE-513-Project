Heart Track: An IoT-enabled Web Application

ECE 513 Project
Create Account or Sign In

Heart Track is a low-cost IoT-enabled web application for monitoring heart rate and blood oxygen saturation levels. It supports real-time data transmission and visualization via a responsive interface.
